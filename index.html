SVM vs MLP: A Machine Learning Model Comparison

Apr 2025 - May 2025

Associated with New Jersey Institute of Technology

I completed a machine learning project comparing Support Vector Machines (SVM) and Multi-Layer Perceptrons (MLP) for handwritten digit classification using the MNIST dataset of 70,000 images. Using 5-fold stratified cross-validation, I evaluated model accuracy, balanced accuracy, and training time across multiple SVM regularization values and MLP hidden layer and learning rate configurations. The project showed that both models performed well, while MLP achieved significantly faster training times than SVM, highlighting the tradeoff between accuracy and computational efficiency.
